Installation

Prerequisites

Make sure you have the following installed on your development machine:

  • Node.js: v18.0.0 or later
  • npm: v10.8.1
  • Android Studio: v2023.3.1.19 (Electric Eel)
  • Java: openjdk 17.0.11
  • GStreamer: gstreamer-1.0-android-universal-1.20.6

Steps

1. Clone the Repository:

  git clone https://github.com/BotLabDynamics/FlightCore.git
  cd FlightCore

2. Install Dependencies:

  npm install

3. Run on Android:

  npx react-native run-android
  npm start

Ensure your Android device is connected or an emulator is running.
