Module src/screens/SettingScreen

Description

The SettingScreen component displays a two-pane layout:

  • A left sidebar (20vw) that lists various settings categories such as General, Calibration, Camera, Offline Maps, Fail-Safe, Media, Instructions, and About. Each category is represented by a focused/unfocused icon and a label. Tapping a category updates the currently selected tab.
  • A right content area (80vw) that dynamically renders the corresponding settings component based on the selected tab. The component is wrapped in a MediaPlayerProvider to enable media playback functionalities in the settings. The settings options are defined in an array (settingsData) that maps each category name to its focused/unfocused icons and the respective component to be rendered. The component utilizes React state to track the active tab and navigates back using the provided navigation prop.

Example

<SettingScreen navigation={navigation} />

Param: navigation

The navigation prop.

Returns

The SettingScreen component.
