Module src/redux/features/SavedMissionSlice

Description

This slice handles the state of saved missions in the application. It provides reducers to save, remove, and set saved missions. The slice exports the following:

  • Actions: saveMission, removeMission, and setSavedMissions
  • Selector: selectSavedMissions to retrieve the saved missions from the Redux store.

Example

// Save a mission:
dispatch(saveMission(mission));
// Remove a mission:
dispatch(removeMission(index));
// Access saved missions in a component:
const savedMissions = useSelector(selectSavedMissions);

Returns

The savedMissionSlice object with actions and selectors.
