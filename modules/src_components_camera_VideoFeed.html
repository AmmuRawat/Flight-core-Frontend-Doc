Module src/components/camera/VideoFeed

Description

  • Retrieves camera settings and flight data (e.g., pitch, roll, yaw, altitude, location) via custom hooks and Redux.
  • Initializes zoom level and scale using Animated values and updates these when the default zoom setting changes.
  • Renders a zoomable video feed using ReactNativeZoomableView that intercepts single tap events.
  • On a single tap, calculates the corresponding image pixel coordinates using device dimensions, intrinsic camera parameters, and flight data.
  • Computes object distance and geographic coordinates from the tapped pixel location using helper functions from a coordinate calculator.
  • Displays a confirmation modal (via ModalOpener) to allow the user to center the clicked object on the map.
  • Sends Mavlink commands to update camera settings when the zoom level is adjusted or when the user confirms an object selection.
  • Integrates a ZoomControls component for user-triggered zoom in/out actions.

Example

import VideoFeed from '@/components/camera/VideoFeed'; // Import the VideoFeed component
<VideoFeed /> // Render the VideoFeed component
