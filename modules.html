Renders an input field for configuring a setting with a heading, description, and live update via Redux.

  • Accepts properties including:
    • heading: The title of the setting.
    • description: A brief explanation of the setting.
    • func: A Redux action creator that is dispatched with the updated value.
    • currentState: The current value of the setting.
    • isChild: A boolean that indicates if the input is a child element (affecting styling).
    • keyboardType: The keyboard type to be used by the TextInput.
  • Maintains internal state for the input text and a boolean for focus state.
  • Updates the local state and dispatches the Redux action on text change.
  • Changes border color when focused.
